🧠 Memory Architecture mem0.ai · Apr 2026
Mem0 Multi-signal Retrieval — Semantic + BM25 + Entity matching fused
Pass 1: Semantic vector similarity — traditional embedding search, recall broad context
Pass 2: BM25 keyword matching — exact term recall, critical cho PII, names, IDs
Pass 3: Entity entity store riêng {collection}_entities, boost memories chứa matched entities
↳ Giải pháp cho "embedding model fallback không match dimension" anh raise trong #leader-tech — multi-signal không rely on single embedding
