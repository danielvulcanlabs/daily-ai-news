Cloudflare Agent Memory: Multi-Channel Retrieval Architecture
  • Ingestion pipeline: extract → verify (8 checks) → classify (fact/event/instruction/task) → vectorize async
  • 5 retrieval channels parallel: FTS + exact fact-key + raw message + direct vector + HyDE vector, fused via Reciprocal Rank Fusion
  • HyDE (Hypothetical Document Embedding) giải quyết vocabulary mismatch giữa query và answer — đặc biệt hiệu quả cho multi-hop queries
