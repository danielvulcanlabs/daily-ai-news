AI Self-Improvement · Research Anthropic Institute
Kiến trúc Recursive Self-Improvement: 3 giai đoạn và điểm ngưỡng nguy hiểm
  • Giai đoạn 1 (AI-assisted): AI viết code theo instruction của human engineers — đang phổ biến rộng rãi. Giai đoạn 2 (AI-led R&D): AI tự đặt research agenda và chạy experiments — Anthropic xác nhận đang ở đây.
  • Giai đoạn 3 (full RSI): AI tự cải thiện training pipeline, kiến trúc, và alignment techniques mà không cần human oversight — đây là điểm ngưỡng cần "brake pedal" toàn cầu.
  • Rủi ro chính: feedback loop acceleration — mỗi thế hệ AI tốt hơn làm AI thế hệ sau được tạo ra nhanh hơn, thu ngắn thời gian để implement safety measures.

Agent Memory · Production mem0.ai
State of AI Agent Memory 2026: Memory đang từ feature thành engineering discipline
  • Memory layer architecture phân kỳ thành ba tier rõ rệt: in-context (short-term), vector store (semantic retrieval), và structured graph (relational memory) — mỗi tier có benchmark riêng.
  • Production bottleneck chính là memory staleness: thông tin cũ trong long-term store gây hallucination tăng 23% so với agent không có memory — cần TTL và invalidation strategy.
  • mem0 (39,993 stars, +144/28d) đang trở thành de-facto standard cho personalized memory layer trong production agent deployments.
