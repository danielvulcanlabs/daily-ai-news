Technical Deep-dive
Agentic Memory mem0.ai
Multi-signal Memory Retrieval: Fusing Semantic + BM25 + Entity Matching
  • Thuật toán mới của Mem0 chạy 3 passes song song: semantic_sim, bm25_keyword, entity_match — kết quả được normalize và fuse thành single score trước khi inject vào context
  • Single-pass ADD-only extraction: agent-generated facts (confirmations, recommendations) được store với equal weight so với user-stated facts — đóng blind spot lớn của memory pipelines trước đây
  • Production requirements đã ship: async_mode=True default, reranking (Cohere/HuggingFace/LLM-based), metadata filtering, memory depth config per project/use-case

Multi-Agent Cost arxiv.org
RecursiveMAS: Latent-Space Agent Collaboration thay thế Text-Based Communication
  • Module RecursiveLink kết nối heterogeneous agents thành collaboration loop — cho phép in-distribution latent thought generation và cross-agent state transfer mà không cần serialize ra text
  • Inner-outer loop learning algorithm: gradient-based credit assignment được share qua recursion rounds — stable gradients trong training, runtime complexity thấp hơn text-based MAS
  • Validated trên 9 benchmarks (math, science, medicine, search, code), 4 collaboration patterns — consistent 8.3% accuracy gain kèm 1.2×–2.4× speedup
